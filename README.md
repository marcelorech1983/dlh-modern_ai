# dlh-modern_ai

Modern AI / applied data science curriculum: data preparation, visualization, and the modeling workflow that follows it. Each project lives in its own folder with its own task list.

## Structure

```
dlh-modern_ai/
└── data_analysis/
    └── data_preparation_visualization/   1/18
```

## Requirements

- Python 3.11
- pycodestyle 2.14.0
- numpy 2.0.2, pandas 2.2.2, scikit-learn 1.6.1, matplotlib 3.10.0, seaborn 0.13.2, scipy 1.16.0, pillow 11.3.0
- Every script starts with `#!/usr/bin/env python3` and is executable
- Every module, class, and function is documented

```bash
pip install -r requirements.txt
```
