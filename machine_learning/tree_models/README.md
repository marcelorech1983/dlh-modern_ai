# Tree-Based Models

Tree-based classification pipeline on the Wine dataset: building, training, and evaluating decision trees, visualizing and reading tree structure, pre- and post-pruning (cost-complexity), random forests with feature importance, and comparing boosting algorithms (AdaBoost, Gradient Boosting, XGBoost, LightGBM).
